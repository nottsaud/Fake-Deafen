## What It Does

FakeDeafen lets you appear deafened to everyone in a voice channel while you continue hearing all audio normally.

## Requirements

- BetterDiscord installed
- Active internet connection
- A valid license key

---

## Installation

1. Contact @overratedsaud on Discord to purchase a key.
2. Drop the plugin file into your BetterDiscord plugins folder.
   - Windows: `%appdata%\BetterDiscord\plugins`
   - macOS: `~/Library/Application Support/BetterDiscord/plugins`
3. Enable the plugin from the Plugins tab in BetterDiscord settings.
4. Open FakeDeafen settings, enter your key, and click Save.
5. Restart the plugin. You are good to go.
